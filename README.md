🧠 Agentes IA — Arquitetura de Agentes Especializados

Repositório dedicado à construção de agentes de IA especializados para engenharia de software.

A proposta deste projeto é organizar prompts, responsabilidades e fluxos de agentes de forma profissional, previsível e escalável.

Ao invés de utilizar um único agente genérico para tudo, este repositório segue uma abordagem baseada em:

especialização de agentes
separação de responsabilidades
contexto limitado
planejamento explícito
execução controlada
arquitetura IA-friendly
🎯 Objetivo

Criar um ecossistema de agentes de IA focados em:

desenvolvimento backend
arquitetura de software
aprendizado técnico
planejamento de features
revisão técnica
análise de problemas
implementação orientada por engenharia

Os agentes foram desenhados para atuar como partes especializadas de um workflow maior.
